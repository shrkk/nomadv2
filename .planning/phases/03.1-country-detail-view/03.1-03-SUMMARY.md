# Phase 03.1 Plan 03: Content Components + CountryDetailSheet Wiring Summary

**One-liner:** Four content components (CityPhotoCarousel, TemperatureNotchPill, StatsPillRow, TripLogCard) built and wired into CountryDetailSheet's ScrollView body, completing the visual content layer of the Country Detail View.

## What Was Built

### Task 1: CityPhotoCarousel and TemperatureNotchPill

**`Nomad/Components/TemperatureNotchPill.swift`** (39 lines):
- `Text(temperature)` with `AppFont.caption()`, `Nomad.cream` foreground
- 28pt height, 12pt horizontal padding
- `Capsule().fill(Nomad.globeBackground.opacity(0.9))` background with 1pt cream stroke at 20% opacity
- Pure display component — no state or logic

**`Nomad/Components/CityPhotoCarousel.swift`** (139 lines):
- `TabView(selection: $selectedCityIndex)` with `.tabViewStyle(.page(indexDisplayMode: .never))`, 320pt height
- Per cluster page: photo (if loaded), `Nomad.warmCard` placeholder with "No photos for {CityName}." if none
- T-3.1-05 auth check: `.denied`/`.restricted` state shows "Allow photo access in Settings." + `Link("Open Settings")` in amber
- Temperature notch overlay via `TemperatureNotchPill` with `.offset(y: -14)` — hidden when temperature is nil
- Location Identity Block below TabView: city name in `AppFont.subheading()` + "Country, City" in `AppFont.body()` at 60% opacity

### Task 2: StatsPillRow, TripLogCard, and CountryDetailSheet Wiring

**`Nomad/Components/StatsPillRow.swift`** (60 lines):
- `HStack(spacing: 0)` with three `statCell` views separated by 1×20pt dividers (`Nomad.globeBackground.opacity(0.15)`)
- Three stats: logs (Int), km (%.1f format), photos (Int)
- Amber values (`AppFont.body()`), gray labels (`AppFont.caption()` at 60% opacity)
- 44pt height, `Nomad.warmCard` background, 22pt corner radius (pill), 16pt horizontal padding

**`Nomad/Components/TripLogCard.swift`** (92 lines):
- `Button(action: onTap)` with `.buttonStyle(.plain)` wrapping card content
- Route strip: 48×36pt via `GeometryReader` + `RoutePreviewPath` in amber stroke; `mappin.circle` fallback when `routePreview.count < 2`
- Trip name (`AppFont.body()`, `.lineLimit(2)`) + formatted date "MMMM d, yyyy" (`AppFont.caption()` at 60% opacity)
- Card: `Nomad.warmCard`, 12pt corner radius, 12pt all-sides padding

**`Nomad/Sheets/CountryDetailSheet.swift`** — body wired with 5 sections:
1. Country header row (from Plan 02)
2. City strip (from Plan 02)
3. `CityPhotoCarousel` (new, `.padding(.top, 16)`)
4. `StatsPillRow` (new, `.padding(.top, 16)`)
5. `tripLogsSection` (new, `.padding(.top, 24).padding(.horizontal, 16)`)

`tripLogsSection` computed property:
- Trips sorted by `startDate` ascending (chronological, oldest first)
- Empty state: "No trips logged for {CityName}." centered, 32pt vertical padding
- Each `TripLogCard` wraps tap in `withAnimation(.spring(response: 0.4, dampingFraction: 0.85))` → `openTripDetail(_:)`
