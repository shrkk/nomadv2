**1. CountryDetailViewModel** (`@Observable @MainActor final class`):
- Properties: `let countryCode: String`, `let countryName: String`, `let allCountryTrips: [TripDocument]`
- State: `var clusters: [CityCluster] = []`, `var selectedCityIndex: Int = 0`, `var isLoading = true`
- Per-cluster resolved state: `var photos: [UUID: UIImage] = [:]` (keyed by CityCluster.id), `var temperatures: [UUID: String?] = [:]` (keyed by CityCluster.id, nil inner value = hide pill, absent key = still loading)
- `var photoCountPerCluster: [UUID: Int] = [:]` (keyed by CityCluster.id — for stats pill)
- `var selectedCluster: CityCluster?` computed from `clusters[safe: selectedCityIndex]` (add safe subscript as private extension on Array)
- `init(countryCode: String, countryName: String, trips: [TripDocument])`
- `func load() async`:
  1. `clusters = clusterTripsByProximity(allCountryTrips)`
  2. `isLoading = false`
  3. Use `withTaskGroup` to call `loadCluster(_:)` for each cluster concurrently.
- `private func loadCluster(_ cluster: CityCluster) async`:
  - Fetch representative photo: check `PHPhotoLibrary.authorizationStatus(for: .readWrite)`. If `.authorized` or `.limited`, create `PHFetchOptions` with predicate `creationDate >= cluster.dateRange.start AND creationDate <= cluster.dateRange.end`, sortDescriptors by creationDate ascending, fetchLimit 1. Use `PHAsset.fetchAssets(with: .image, options:)`. If result has firstObject, request image using `PHImageManager.default().requestImage` with targetSize `CGSize(width: UIScreen.main.bounds.width * 3, height: 960)`, contentMode `.aspectFill`. Use `withCheckedContinuation` (same pattern as PhotoGalleryStrip.swift). Set `photos[cluster.id] = image`.
  - Also count total photos for stats: run a second fetch without fetchLimit and set `photoCountPerCluster[cluster.id] = assets.count`.
  - Fetch temperature: `let temp = await NomadWeatherService.shared.fetchTemperature(for: cluster.centroid)`. Set `temperatures[cluster.id] = temp` (nil if unavailable).
