# Phase 03.3 Plan 01: ViewModel All-Photos + CityPhotoCarousel Multi-Photo Summary

**One-liner:** ViewModel upgraded from single-representative-photo to `cityPhotos: [UUID: [UIImage]]` with GPS bounding box fetching and memory eviction; CityPhotoCarousel rewritten as single-cluster inner `TabView` with `@State photoIndex` and page indicator dots.

## What Was Built

**Task 1 — ViewModel photo data layer:**

- `var cityPhotos: [UUID: [UIImage]] = [:]` replaces the old `var photos: [UUID: UIImage] = [:]`
- `private func boundingBox(for cluster: CityCluster)` computes min/max lat/lon from all `routePreview` coordinates across a cluster's trips
- `loadCluster()` photo fetch rewritten: no `fetchLimit`, enumerate all PHAssets in date range, apply GPS bounding box filter; nil-location assets fall through to date-range-only match (DETAIL-04); images loaded with `.opportunistic` delivery mode
- `func evictDistantPhotos()` clears `cityPhotos` entries for clusters more than 2 positions away from `selectedCityIndex` (T-3.3-02 DoS mitigation)
- Temporary bridge: `var photos: [UUID: UIImage] { cityPhotos.compactMapValues { $0.first } }` keeps `CityThumbnailCard` call site compiling until Plan 02

**Task 2 — CityPhotoCarousel multi-photo carousel:**

- Parameters changed from `(clusters: [CityCluster], selectedCityIndex: Binding<Int>, photos: [UUID: UIImage], temperatures: [UUID: String?], countryName: String)` to `(cluster: CityCluster, photos: [UIImage], temperature: String?, countryName: String)`
- `@State private var photoIndex: Int = 0` — local per-city, resets when SwiftUI creates a fresh instance per outer TabView page
- Inner `TabView(selection: $photoIndex)` pages between photos within one city
- Page indicator dots: 8pt `Circle()` with `Nomad.accent` fill for active, `Nomad.textSecondary` at 40% opacity for inactive, shown only when `photos.count >= 2` (UI-SPEC)
- `.contentShape(Rectangle())` on photo images enforces view-only (D-05)
- `emptyPhotoPlaceholder` handles `.denied`/`.restricted` auth with Settings link, and no-photos state
