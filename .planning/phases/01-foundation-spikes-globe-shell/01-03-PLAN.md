Create `Nomad/Globe/GlobeCountryOverlay.swift` — converts parsed CountryFeature polygon data into an equirectangular UIImage texture for the globe sphere.

The texture-paint approach (recommended by RESEARCH.md Pattern 2):
1. Create a UIImage canvas at 4096x2048 (equirectangular projection — width is 360 degrees, height is 180 degrees).
2. For each visited country, iterate its polygon coordinate arrays.
3. Convert each `CLLocationCoordinate2D` (lat, lon) to UV pixel coordinates using:
```swift
func geoJSONToPixel(lon: Double, lat: Double, size: CGSize) -> CGPoint {
    let x = ((lon + 180.0) / 360.0) * size.width   // 0...width left to right
    let y = ((90.0 - lat) / 180.0) * size.height    // 0...height top to bottom (flip Y)
    return CGPoint(x: x, y: y)
}
```
4. Draw filled polygons using CoreGraphics `CGMutablePath` and `fillPath()`.
5. Fill color: `UIColor(red: 0.910, green: 0.643, blue: 0.290, alpha: 0.6)` — this is `#E8A44A` at 60% opacity per D-03.
6. Apply outer glow: Before filling each polygon, set `ctx.cgContext.setShadow(offset: .zero, blur: 8, color: UIColor(red: 0.910, green: 0.643, blue: 0.290, alpha: 0.3).cgColor)` — this creates the 2-3px outer glow per D-03.
7. Base canvas is transparent (clear). Only visited countries are drawn.

For Phase 1, hardcode 5 visited countries that do NOT cross the anti-meridian: Japan ("JP"), France ("FR"), Kenya ("KE"), Australia ("AU"), Brazil ("BR").

The function signature:
```swift
struct GlobeCountryOverlay {
    static let hardcodedVisitedCodes: Set<String> = ["JP", "FR", "KE", "AU", "BR"]

    /// Renders visited country polygons onto an equirectangular UIImage.
    /// - Parameters:
    ///   - countries: All parsed country features from GeoJSON
    ///   - visitedCodes: ISO_A2 codes of visited countries
    ///   - size: Texture size (default 4096x2048)
    /// - Returns: UIImage with filled country polygons on transparent background
    static func renderOverlayTexture(
        countries: [CountryFeature],
        visitedCodes: Set<String> = hardcodedVisitedCodes,
        size: CGSize = CGSize(width: 4096, height: 2048)
    ) -> UIImage

    /// Converts overlay UIImage to RealityKit TextureResource
    static func makeTextureResource(from image: UIImage) throws -> TextureResource
}
```

The `makeTextureResource` method:
```swift
static func makeTextureResource(from image: UIImage) throws -> TextureResource {
    guard let cgImage = image.cgImage else {
        throw GlobeError.parseError("Failed to get CGImage from overlay")
    }
    return try TextureResource.generate(
        from: cgImage,
        options: .init(semantic: .color)
    )
}
```

This overlay runs off the main thread (called from GlobeViewModel's async loading path). The rendering at 4096x2048 should take under 200ms.

1. Create `Nomad/Globe/GlobeViewModel.swift` — manages globe state, loads GeoJSON, and prepares the overlay texture:

```swift
import SwiftUI
import RealityKit
import Observation

@Observable
class GlobeViewModel {
    var countries: [CountryFeature] = []
    var overlayTexture: TextureResource?
    var isLoading = true
    var error: String?

    // Globe rotation state (controlled by gestures in GlobeView)
    var rotationX: Float = 0  // pitch
    var rotationY: Float = 0  // yaw
    var cameraDistance: Float = 2.0  // default camera distance from origin

    // Minimum and maximum zoom distances
    let minCameraDistance: Float = 0.8   // continent-level zoom
    let maxCameraDistance: Float = 2.5   // full globe view

    func loadGlobeData() async {
        do {
            let parser = GeoJSONParser()
            let loaded = try await parser.loadCountries()
            countries = loaded

            // Render overlay texture off main thread
            let overlayImage = GlobeCountryOverlay.renderOverlayTexture(countries: loaded)
            let texture = try GlobeCountryOverlay.makeTextureResource(from: overlayImage)

            await MainActor.run {
                overlayTexture = texture
                isLoading = false
            }
        } catch {
            await MainActor.run {
                self.error = error.localizedDescription
                isLoading = false
            }
        }
    }
}
```

2. Create `Nomad/Globe/GlobeView.swift` — RealityView with virtual camera, sphere mesh, country overlay texture, star field, and gestures:

```swift
import SwiftUI
import RealityKit

struct GlobeView: View {
    @State private var viewModel = GlobeViewModel()
    @State private var globeEntity: ModelEntity?
    @State private var textureApplied: Bool = false

    // Gesture tracking
    @State private var lastDragTranslation: CGSize = .zero

    var body: some View {
        ZStack {
            // Deep space background per D-01: #0A0A14
            Color.Nomad.globeBackground
                .ignoresSafeArea()

            if viewModel.isLoading {
                ProgressView()
                    .tint(Color.Nomad.amber)
            } else {
                RealityView { content in
                    content.camera = .virtual

                    // --- Star field per D-01: subtle stars behind globe ---
                    // Create ~80 small sphere entities at high radius to simulate stars.
                    // Star density/size is Claude's discretion per CONTEXT.md.
                    for _ in 0..<80 {
                        let star = ModelEntity(
                            mesh: .generateSphere(radius: Float.random(in: 0.003...0.008)),
                            materials: [UnlitMaterial(color: UIColor(
                                white: 1.0,
                                alpha: CGFloat(Float.random(in: 0.15...0.40))
                            ))]
                        )
                        // Random position on a large sphere (radius 5-8) surrounding the scene
                        let theta = Float.random(in: 0...(2 * .pi))
                        let phi = Float.random(in: 0...(.pi))
                        let r = Float.random(in: 5...8)
                        star.position = SIMD3<Float>(
                            r * sin(phi) * cos(theta),
                            r * cos(phi),
                            r * sin(phi) * sin(theta)
                        )
                        content.add(star)
                    }

                    // Create globe sphere — radius 0.5 in world units
                    let sphere = ModelEntity(
                        mesh: .generateSphere(radius: 0.5),
                        materials: [makeGlobeMaterial()]
                    )
                    sphere.generateCollisionShapes(recursive: false)
                    content.add(sphere)
                    globeEntity = sphere

                    // Add directional light per D-02: warm hemisphere
                    let light = DirectionalLight()
                    light.light.color = UIColor(red: 1.0, green: 0.95, blue: 0.85, alpha: 1.0) // warm tint
                    light.light.intensity = 1000
                    light.look(at: SIMD3(0, 0, 0), from: SIMD3(1, 0.5, 1), relativeTo: nil)
                    content.add(light)

                    // Set camera position
                    let camera = PerspectiveCamera()
                    camera.camera.fieldOfViewInDegrees = 45
                    camera.position = SIMD3(0, 0, viewModel.cameraDistance)
                    content.add(camera)

                } update: { content in
                    // Update globe rotation when gesture changes
                    if let globe = globeEntity {
                        let pitchQuat = simd_quatf(angle: viewModel.rotationX, axis: SIMD3(1, 0, 0))
                        let yawQuat = simd_quatf(angle: viewModel.rotationY, axis: SIMD3(0, 1, 0))
                        globe.orientation = yawQuat * pitchQuat
                    }

                    // Update material if texture becomes available — use local flag to avoid
                    // re-entrant @Observable mutation inside the RealityView update closure.
                    if !textureApplied, let texture = viewModel.overlayTexture, let globe = globeEntity {
                        var material = UnlitMaterial()
                        material.color = .init(
                            tint: .white,
                            texture: MaterialParameters.Texture(texture)
                        )
                        globe.model?.materials = [material]
                        textureApplied = true
                    }
                }
                .gesture(
                    DragGesture()
                        .onChanged { value in
                            let deltaX = Float(value.translation.width - lastDragTranslation.width) * 0.005
                            let deltaY = Float(value.translation.height - lastDragTranslation.height) * 0.005
                            viewModel.rotationY += deltaX
                            viewModel.rotationX += deltaY
                            lastDragTranslation = value.translation
                        }
                        .onEnded { _ in
                            lastDragTranslation = .zero
                        }
                )
                .gesture(
                    MagnifyGesture()
                        .onChanged { value in
                            let scale = Float(value.magnification)
                            let newDistance = viewModel.cameraDistance / scale
                            viewModel.cameraDistance = min(
                                viewModel.maxCameraDistance,
                                max(viewModel.minCameraDistance, newDistance)
                            )
                        }
                )
                .ignoresSafeArea()
            }
        }
        .task {
            await viewModel.loadGlobeData()
        }
    }

    private func makeGlobeMaterial() -> any RealityKit.Material {
        // Initial material — will be replaced with texture overlay once loaded
        var material = UnlitMaterial()
        material.color = .init(tint: UIColor(Color.Nomad.globeBackground))
        return material
    }
}
```

KEY IMPLEMENTATION NOTES:
- `content.camera = .virtual` — this is critical. Without it, RealityView may attempt an AR session. Per RESEARCH.md Pitfall 2, default ARView activates camera feed.
- Sphere radius is 0.5 world units. Camera at distance 2.0 shows full globe.
- DragGesture maps to rotation quaternions applied to the globe entity, not the camera. This means the globe spins under the user's finger.
- MagnifyGesture adjusts camera distance between 0.8 (continent zoom) and 2.5 (full globe).
- The overlay texture is applied as an `UnlitMaterial` so country highlights render at their exact color values without lighting interference. The directional light (D-02) affects only the base globe appearance; the overlay sits on top.
- The country polygon overlay is rendered once at load time, not per frame. Static per D-04.
- **Star field per D-01:** ~80 small unlit sphere entities placed randomly on a large surrounding sphere (radius 5-8). White color at 15-40% opacity. Star density and size are Claude's discretion per CONTEXT.md. This creates the "dark space with subtle stars" look.
- **CRITICAL: Texture application uses a local `@State var textureApplied: Bool` flag instead of nulling `viewModel.overlayTexture` inside the RealityView update closure.** Mutating an @Observable property inside `update` can trigger re-entrant update calls or MainActor violations. The local @State flag avoids this.
